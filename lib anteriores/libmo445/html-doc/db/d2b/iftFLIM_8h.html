iftFLIM.h File Reference

Description

FLIM: Feature Learning From Image Markers – functions to learn the parameters of a sequence of convolutional layers from image markers and employ the resulting model for feature extraction.

Note
Programs:
  • iftFLIM-LearnModel.c = Learns the CNN model
  • iftFLIM-ExtractFeatures.c = Extracts image features.
