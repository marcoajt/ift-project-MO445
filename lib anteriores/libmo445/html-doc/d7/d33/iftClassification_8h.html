void iftClassifyByOneClassOPF ( iftDataSet *  Ztest,
const iftKnnGraph *  graph,
float  quantile_of_k

Classify by One Class OPF.

If a testing sample is inside the adjacency radius of any training sample, it belongs this training sample's cluster and its label is IFT_POSITIVE_CLASS Otherwise, it is an outlier (status is IFT_OUTLIER) and its predicted label is IFT_NEGATIVE_CLASS.

Each training sample s has its own adjacency radius which corresponds to its distance to its i-th nearest neighbor (i in [1, graph->k]) which is defined as:

i = ceil(quantile_of_k * graph->k)

where graph->k is the number of neighbors of each training sample, and quantile_of_k is the quantile used to define the rank of i.

quantile_of_k = 0.0 ==> we consider the nearest neighbor, quantile_of_k = 0.5 ==> the median value, quantile_of_k = 1.0 ==> the k-th nearest neighbor (the farthest one).

Note
Only the samples with status IFT_TEST in Ztest will be classified.
Parameters
ZtestDataset for classification (only the samples with status IFT_TEST will be classified)
graphTrained KnnGraph (by iftUnsupTrain) with only the training samples of the target class.
quantile_of_kQuantile used to find the rank of the i-th nearest neighbor that defines the adjacency radius of each training sample. It must be in [0, 1]

Classifies testing samples (status = IFT_TEST) from the dataset Ztest by using Supervised OPF and computes a certainty rate for each testing sample.

Author
Samuel Martins
Date
Jun 14, 2016

Similar to iftClassify(), this function also computes the certainty value that the test samples belong to their predicted labels.
For that, the test sample is initially "conquered" by the train. sample that offered the minimum cost min_cost1. Suppose that its label is A.
Then, the function searches another class/truelabel different from A whose training sample offers the minimum cost min_cost2 to the testing sample.
Finally, the certainty value is computed by:
certainty = (min_cost2) / (min_cost1 + min_cost2)

Note
The certainty value of the testing sample after classification is stored as its weight in the dataset.
If a testing sample is classified incorrectly, its status will be IFT_ERROR.
Parameters
graphSupervised OPF (Complete Graph).
ZtestDataset with the testing samples to be classified.
Returns
Number of classification errors.
