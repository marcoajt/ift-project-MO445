iftBoundingBox ift_image_tile::bb

Bounding box of the tiles in the original coordinate system. The tiles are computed only for the given bounding box, which encompasses the entire image when necessary.

iftBoundingBox* ift_image_tile::tile_coords

The bounding boxes of each tile RELATIVE TO THE START OF THE ORIGINAL BOUNDING BOX. That is, the first tile will have coordinates (0,0,0) relative to voxel bb.min, which may or may not coincide with the first voxel of the original image.
